Stack Calculator (Go lang sample)
===========

Go言語で書かれたサンプルプログラムのスタック計算機です。

How to use
----------

make コマンドでビルド出来ます。
./gost で起動します。
プロンプト GOST> が表示されますので計算式を入力します。
計算式には，数字，演算子(+-*/)を空白もしくは改行を区切りとして入力します。
また，ピリオド(.) を入力すると，スタックトップを取り出し画面に数値を表示します。

例）
GOST> 2 3 4 - * .
と入力すると，画面に -2 と表示されます。
